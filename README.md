## ✨ Features

PDF Master Pro is a comprehensive PDF solution with **16+ core features**, all processed **entirely offline** for maximum privacy and speed.

| Category | Features |
|----------|----------|
| **📖 Reading** | PDF Reader, Recent Files, Favorites |
| **✏️ Editing** | Merge PDF, Split PDF, Rotate PDF, Reorder Pages, Extract Pages |
| **🔒 Security** | Lock PDF (password), Unlock PDF, Watermark |
| **📸 Input** | Image to PDF, Document Scanner (CameraX) |
| **📦 Utility** | Compress PDF, Signature, Material Design 3 UI |

> ⚡ **100% offline** – no data ever leaves your device.

## 🛠 Tech Stack

| Technology | Purpose |
|------------|---------|
| **Java** | Core language |
| **XML** | UI layouts |
| **Room** | Local database (favorites, recent) |
| **Navigation** | Fragment navigation |
| **RecyclerView** | Lists & grids |
| **Material Design 3** | Modern UI components |
| **CameraX** | Document scanning |
| **PdfRenderer** | PDF rendering engine |
| **ViewBinding** | Type-safe view access |
| **WorkManager** | Background tasks (compression) |

---

## 🏗 Architecture

The app follows a **clean MVVM architecture** with repository pattern:


**Core Engines:**
- **Reader Engine** – PdfRenderer + custom rendering pipeline
- **Scanner Engine** – CameraX + image processing
- **Utilities** – Compression, merging, splitting, watermarking
